🔑 获取 Trello API Token

步骤 1: 输入您的 API Key

从 https://trello.com/app-key 复制您的 API Key

步骤 2: 生成授权链接

点击下面的按钮进行授权：

步骤 3: 保存您的 Token

授权后，Trello 会显示一个 Token，复制并保存到安全的地方

重要： Token 相当于密码，请妥善保管，不要分享给他人

步骤 4: 配置环境变量

在 backend/.env 文件中添加：

TRELLO_API_KEY=your_api_key TRELLO_API_TOKEN=your_token_here TRELLO_BOARD_ID=hSqfthYt
✅ 授权链接已生成！点击上面的授权按钮完成授权。
